# 商品销售分析计算指标

## 使用原则

- 本文档描述“指标怎么计算”，不写具体 SQL 细节。
- 本文档中的指标应与 `metrics.md` 保持一致。
- 本文档统一使用 `÷` 表示除法；符号 `/` 只用于并列指标名或路径分隔，不表示除法。
- 所有比率、占比、均值类指标在分母为 0、为空或样本量过小时，不应强行计算，应返回空值或标记“样本不足”。
- 如果数据库缺少某个指标所需字段，不要臆造字段，不要强行计算。
- 如果计算结果依赖估算字段，应在结果中标记“估算口径”。

## 统一计算口径

- 默认只纳入“已支付 / 已完成购买”的订单。
- 默认排除已取消订单、未支付订单、测试订单、内部订单和明显重复订单。
- 默认使用购买完成时间作为统计时间，例如支付完成时间或订单完成时间。
- 默认商品销售收入不含税费、运费和平台附加费；如果用户明确要求 GMV、总收款或含税成交额，才切换口径。
- 退款默认独立记录，并回冲销售结果。
- 成本优先使用成交当时商品成本；如果使用标准成本或最近成本，应标记为估算口径。
- 商品级分析优先使用订单商品明细粒度，订单级分析使用订单粒度，客户分析使用客户与订单粒度。
- 周期对比必须保证时间窗口、订单口径、对象范围一致。

## 中间量说明

以下不是最终分析指标，只是计算其他指标时可能需要的中间量。

- 销货成本（cogs）
  - 计算公式：销货成本 = sum(成交时单位成本 × 净售出件数)
  - 用途：用于计算毛利润、毛利率、库存周转次数。
  - 缺字段降级：没有成本字段时，不计算毛利润、毛利率、贡献利润、净利润和库存周转次数。
  - 注意：如果成本不是成交当时成本，应标记为估算口径。

- 净售出件数（net_units_sold）
  - 计算公式：净售出件数 = 售出件数 - 退货件数
  - 用途：用于计算成本、毛利和退货相关指标。
  - 缺字段降级：没有退货件数字段时，可使用售出件数近似，但需要标记“未扣除退货件数”。

- 销售折让金额（allowance_amount）
  - 计算公式：销售折让金额 = sum(成交后给予客户的价格减免或补偿)
  - 用途：用于计算销售净额。
  - 缺字段降级：没有销售折让字段时，默认记为 0。
  - 注意：如果存在价保补差、售后补偿等字段，应先确认是否属于销售折让口径。

- 平均库存成本（avg_inventory_cost）
  - 计算公式：平均库存成本 = 日库存成本快照平均值；如果没有日快照，可用周快照平均值；如果都没有，可用（期初库存成本 + 期末库存成本）÷ 2。
  - 用途：用于计算库存周转次数和库存可支撑天数。
  - 缺字段降级：没有库存快照或期初期末库存成本时，不计算库存周转次数和成本口径库存可支撑天数。

- 购买商品件数（items_purchased）
  - 计算公式：购买商品件数 = sum(购买事件中的商品数量)
  - 用途：用于计算结账到购买转化率、浏览到购买转化率。
  - 缺字段降级：没有购买商品数量时，可用购买次数近似，但需要说明口径变化。

- RFM 基础变量
  - 最近购买间隔（recency）= 分析截止日期 - 客户最近一次下单日期。
  - 购买频次（frequency）= 客户累计订单数。
  - 消费金额（monetary）= 客户累计销售净额。
  - 用途：用于生成 RFM 客户分层。
  - 注意：RFM 基础变量通常不单独作为最终指标输出，除非用户明确询问。

## 全渠道核心经营指标

- 销售总额（gross_sales）[核心]
  - 计算公式：销售总额 = sum(折扣前商品单价 × 售出数量)
  - 缺字段降级：如果没有折扣前价格，只能使用成交价近似，并说明这不是严格的销售总额。
  - 注意：如果只有订单总金额、没有商品行明细，只能做订单级销售总额，不能做商品级销售总额。

- 销售净额（net_sales）[核心]
  - 计算公式：销售净额 = 销售总额 - 折扣金额 - 退款金额 - 销售折让金额
  - 缺字段降级：没有退款字段时，销售净额应说明“未扣除退款”；没有折扣字段时，应说明折扣口径缺失。
  - 注意：如果数据库已有销售净额字段，应先确认它是否已扣除折扣、退款和销售折让。

- 订单数（orders）[核心]
  - 计算公式：订单数 = count(distinct 订单 ID)
  - 缺字段降级：没有订单 ID 时，不能计算订单数。
  - 注意：如果从订单商品明细表计算订单数，必须去重，不能直接统计行数。

- 售出件数（units_sold）[核心]
  - 计算公式：售出件数 = sum(售出数量)
  - 缺字段降级：没有数量字段时，如果一行确定代表一件商品，可用行数近似，并说明口径。
  - 注意：如果一行可能代表多件商品，不能用行数代替售出件数。

- 去重购买客户数（unique_customers）[核心]
  - 计算公式：去重购买客户数 = count(distinct 客户 ID)
  - 缺字段降级：没有客户 ID 时，不计算该指标。
  - 注意：如果存在游客客户或匿名客户，应说明是否纳入统计。

- 毛利润（gross_profit）[条件]
  - 计算公式：毛利润 = 销售净额 - 销货成本
  - 缺字段降级：没有成本字段时，不计算毛利润。
  - 注意：如果成本不是成交当时成本，应标记为估算口径。

- 毛利率（gross_margin）[条件]
  - 计算公式：毛利率 = 毛利润 ÷ 销售净额
  - 缺字段降级：没有成本字段或销售净额为 0 时，不计算毛利率。
  - 注意：毛利率必须与毛利润使用同一成本口径。

- 贡献利润（contribution_profit）[条件]
  - 计算公式：贡献利润 = 毛利润 - 履约成本 - 支付手续费 - 平台费用 - 可归因广告花费 - 其他直接变动成本
  - 缺字段降级：如果只有毛利润、缺少后续可归因成本，不计算贡献利润。
  - 注意：如果只扣除了部分成本，应写明“当前可归因成本口径下的贡献利润”，不要命名为净利润。

- 净利润（net_profit）[条件]
  - 计算公式：净利润 = 毛利润 - 经营费用 - 利息费用 - 税费
  - 缺字段降级：缺少完整经营费用、利息或税费时，不计算净利润。
  - 注意：不要把毛利润或贡献利润误称为净利润。

## 价格与客单指标

- 总额口径客单价（aov_gross，Average Order Value，平均订单金额）[核心]
  - 计算公式：总额口径客单价 = 销售总额 ÷ 订单数
  - 缺字段降级：没有销售总额或订单数为 0 时，不计算该指标。
  - 注意：该指标未扣除折扣、退款和销售折让，适合观察原始成交规模。

- 净额口径客单价（aov_net，Average Order Value，平均订单金额）[核心]
  - 计算公式：净额口径客单价 = 销售净额 ÷ 订单数
  - 缺字段降级：没有销售净额或订单数为 0 时，不计算该指标。
  - 注意：这是默认客单价指标，应优先使用。

- 净额口径平均售价（asp_net，Average Selling Price，平均销售单价）[核心]
  - 计算公式：净额口径平均售价 = 销售净额 ÷ 售出件数
  - 缺字段降级：没有售出件数或售出件数为 0 时，不计算该指标。
  - 注意：如果销售净额只有订单级金额，不能直接计算商品级平均售价，除非可以合理分摊到商品行。

- 折扣金额（discount_amount）[核心]
  - 计算公式：折扣金额 = sum(商品行折扣金额 + 分摊到商品行的订单级折扣金额)
  - 缺字段降级：如果只有订单级折扣，只能做订单级折扣分析；如果没有分摊规则，不要强行计算 SKU 级折扣。
  - 注意：折扣金额应与销售总额使用同一统计范围。

- 折扣率（discount_rate）[核心]
  - 计算公式：折扣率 = 折扣金额 ÷ 销售总额
  - 缺字段降级：没有折扣金额或销售总额为 0 时，不计算该指标。
  - 注意：折扣率较高时，需要结合销售净额和毛利率判断是否存在让利换量。

- 退款金额（refund_amount）[核心]
  - 计算公式：退款金额 = sum(退款金额)
  - 缺字段降级：没有退款字段或退款表时，不计算该指标。
  - 注意：如果退款只记录订单级金额，不要强行拆分到 SKU 级。

- 金额口径退款率（refund_rate_amt）[核心]
  - 计算公式：金额口径退款率 = 退款金额 ÷ 销售总额
  - 缺字段降级：没有退款金额或销售总额为 0 时，不计算该指标。
  - 注意：该指标用于衡量退款对销售结果的侵蚀。

- 件数口径退货率（return_rate_units）[条件]
  - 计算公式：件数口径退货率 = 退货件数 ÷ 售出件数
  - 缺字段降级：没有退货件数字段或售出件数为 0 时，不计算该指标。
  - 注意：该指标更适合定位商品质量、描述偏差或用户预期不符问题。

## 商品结构指标

- 单品销售贡献占比（sku_sales_share，SKU 指 Stock Keeping Unit，即库存量单位/单品编码）[核心]
  - 计算公式：单品销售贡献占比 = 单品销售净额 ÷ 总销售净额
  - 缺字段降级：没有 SKU 或商品 ID 时，不计算该指标；没有销售净额时，可用成交金额替代并说明口径。
  - 注意：用于判断哪些单品拉动或拖累整体销售。

- 类目销售贡献占比（category_sales_share）[核心]
  - 计算公式：类目销售贡献占比 = 类目销售净额 ÷ 总销售净额
  - 缺字段降级：没有类目字段时，不计算该指标。
  - 注意：如果存在多级类目，应说明使用的是一级类目、二级类目还是末级类目。

- 品牌销售贡献占比（brand_sales_share）[条件]
  - 计算公式：品牌销售贡献占比 = 品牌销售净额 ÷ 总销售净额
  - 缺字段降级：没有品牌字段时，不计算该指标。
  - 注意：用于观察品牌结构变化，不应只看销售额，也应结合毛利或退货情况。

- 单品毛利排序（sku_margin_rank，SKU 指库存量单位/单品编码）[条件]
  - 计算公式：单品毛利润 = 单品销售净额 - 单品销货成本；单品毛利率 = 单品毛利润 ÷ 单品销售净额
  - 缺字段降级：没有成本字段时，不计算单品毛利排序。
  - 注意：排序可按毛利润或毛利率进行，但应同时保留销量和销售额，避免只看高毛利率小样本商品。

- 商品结构变化（product_mix_change）[核心]
  - 计算公式：商品结构变化 = 当前周期商品/类目销售占比 - 对比周期商品/类目销售占比
  - 缺字段降级：没有对比周期时，只能描述当前结构，不能判断结构变化。
  - 注意：如果有成本字段，可进一步判断结构是否转向高毛利或低毛利商品；没有成本字段时，只能判断价格或销售结构变化。

## 渠道 / 门店 / 地区指标

- 渠道销售额 / 门店销售额 / 地区销售额（channel_sales / store_sales / region_sales）[核心]
  - 计算公式：渠道销售额 = 按渠道汇总销售净额；门店销售额 = 按门店汇总销售净额；地区销售额 = 按地区汇总销售净额
  - 缺字段降级：缺少哪个维度，就不计算哪个维度的销售额。
  - 注意：如果只有订单级渠道或地区，只能做订单级维度分析，不能直接推导商品级渠道或地区表现。

- 渠道毛利 / 门店毛利（channel_margin / store_margin）[条件]
  - 计算公式：渠道毛利率 = 渠道毛利润 ÷ 渠道销售净额；门店毛利率 = 门店毛利润 ÷ 门店销售净额
  - 缺字段降级：没有成本字段时，不计算渠道毛利或门店毛利。
  - 注意：渠道或门店的销售规模高，不代表盈利质量一定高，应结合毛利率判断。

- 渠道订单占比（channel_order_share）[核心]
  - 计算公式：渠道订单占比 = 渠道订单数 ÷ 总订单数
  - 缺字段降级：没有渠道字段或订单 ID 时，不计算该指标。
  - 注意：如果一笔订单可能跨多个渠道，必须先确认业务归属规则，避免重复计数。

- 可比门店销售增长率（same_store_sales_growth，同店销售增长率）[条件]
  - 计算公式：可比门店销售增长率 =（当前周期可比门店销售净额 - 对比周期同一批可比门店销售净额）÷ 对比周期同一批可比门店销售净额
  - 缺字段降级：无法确认可比门店范围、门店开业时间或对比周期销售额为 0 时，不计算该指标。
  - 注意：可比门店应满足当前周期与对比周期都开业、营业时长满足要求、无重大关停/搬迁/改造影响。

## 线上漏斗指标

- 商品浏览量（items_viewed）[条件]
  - 计算公式：商品浏览量 = sum(商品浏览事件中的商品数量)
  - 缺字段降级：没有事件埋点表时，不计算该指标；没有商品数量时，可用浏览事件次数近似并说明口径。
  - 注意：应保证浏览事件与后续加购、结账、购买事件处于同一统计窗口。

- 加购商品数（items_added_to_cart）[条件]
  - 计算公式：加购商品数 = sum(加购事件中的商品数量)
  - 缺字段降级：没有加购事件时，不计算该指标；没有商品数量时，可用加购事件次数近似并说明口径。
  - 注意：加购商品数用于衡量购买意向，不等同于订单数。

- 进入结账商品数（items_checked_out）[条件]
  - 计算公式：进入结账商品数 = sum(进入结账事件中的商品数量)
  - 缺字段降级：没有结账事件时，不计算该指标。
  - 注意：该指标用于定位购物车到支付前的流失。

- 购买次数（purchases）[条件]
  - 计算公式：购买次数 = count(购买事件)
  - 缺字段降级：没有事件表时，可用订单数近似，但要说明口径变化。
  - 注意：购买次数来自事件表，订单数来自订单表，两者可能因埋点丢失而不一致。

- 浏览到加购转化率（view_to_cart_rate）[条件]
  - 计算公式：浏览到加购转化率 = 加购商品数 ÷ 商品浏览量
  - 缺字段降级：商品浏览量为 0 或事件粒度不一致时，不计算该指标。
  - 注意：该指标用于判断商品详情页、价格和商品吸引力。

- 加购到结账转化率（cart_to_checkout_rate）[条件]
  - 计算公式：加购到结账转化率 = 进入结账商品数 ÷ 加购商品数
  - 缺字段降级：加购商品数为 0 时，不计算该指标。
  - 注意：该指标用于判断购物车阶段是否存在损耗。

- 结账到购买转化率（checkout_to_purchase_rate）[条件]
  - 计算公式：结账到购买转化率 = 购买商品件数 ÷ 进入结账商品数
  - 缺字段降级：进入结账商品数为 0，或没有购买事件时，不计算该指标。
  - 注意：该指标用于判断支付流程、运费、库存锁定、履约承诺等问题。

- 浏览到购买转化率（view_to_purchase_rate）[条件]
  - 计算公式：浏览到购买转化率 = 购买商品件数 ÷ 商品浏览量
  - 缺字段降级：商品浏览量为 0，或事件链路不完整时，不计算该指标。
  - 注意：该指标是商品级综合转化效率指标。

## 促销指标

- 促销曝光量 / 促销点击量（promo_views / promo_clicks）[条件]
  - 计算公式：促销曝光量 = count(促销曝光事件)；促销点击量 = count(促销点击事件)
  - 缺字段降级：没有促销曝光或点击事件时，不计算该指标。
  - 注意：如果只有促销订单数据，只能分析促销产出，不能分析促销触达。

- 促销订单数 / 促销销售额（promo_orders / promo_sales）[条件]
  - 计算公式：促销订单数 = count(distinct 促销相关订单 ID)；促销销售额 = sum(促销相关销售净额)
  - 缺字段降级：没有促销标识、促销订单关联关系或优惠券使用记录时，不计算该指标。
  - 注意：如果使用优惠券订单近似促销订单，需要说明促销口径。

- 促销销售占比（promo_sales_share）[条件]
  - 计算公式：促销销售占比 = 促销销售额 ÷ 总销售净额
  - 缺字段降级：没有促销销售额或总销售净额为 0 时，不计算该指标。
  - 注意：用于判断整体销售结果对促销活动的依赖程度。

- 促销毛利影响（promo_margin_impact）[条件]
  - 计算公式：促销毛利影响 = 促销毛利率 - 非促销毛利率；其中，促销毛利率 = 促销毛利润 ÷ 促销销售净额，非促销毛利率 = 非促销毛利润 ÷ 非促销销售净额。
  - 缺字段降级：没有成本字段、促销标识或非促销对照组时，不计算该指标。
  - 注意：没有对照组时，只能描述促销订单毛利，不要写“促销影响”。

## 库存与供应链指标

- 期末库存件数 / 期末库存成本（closing_inventory_units / closing_inventory_cost）[条件]
  - 计算公式：期末库存件数 = 周期结束时在库件数；期末库存成本 = 周期结束时在库库存成本。
  - 缺字段降级：没有库存快照时，不计算该指标；如果只有当前库存，只能输出当前库存，不能输出指定周期期末库存。
  - 注意：库存时间点必须与分析周期对齐。

- 库存周转次数（inventory_turnover）[条件]
  - 计算公式：库存周转次数 = 销货成本 ÷ 平均库存成本
  - 缺字段降级：没有成本字段或平均库存成本时，不计算该指标。
  - 注意：库存周转次数用于衡量库存效率，不能单独判断商品好坏。

- 售罄率 / 动销消化率（sell_through_rate）[条件]
  - 计算公式：售罄率 = 售出数量 ÷ 收货数量
  - 缺字段降级：没有收货数量或收货数量为 0 时，不计算该指标。
  - 注意：适合用于新品、批次商品或明确进货周期的商品分析。

- 库存可支撑天数（days_on_hand，DOH，Days on Hand）[条件]
  - 计算公式：成本口径库存可支撑天数 = 平均库存成本 ÷ 销货成本 × 观察周期天数；件数口径库存可支撑天数 = 期末库存件数 ÷ 平均日销量。
  - 缺字段降级：没有库存快照时，不计算该指标；平均日销量为 0 时，不计算件数口径库存可支撑天数。
  - 注意：如果成本字段完整，优先使用成本口径；否则可使用件数口径并说明。

- 缺货天数 / 缺货率（stockout_days / stockout_rate）[条件]
  - 计算公式：缺货天数 = count(库存数量 <= 0 的日期)；缺货率 = 缺货天数 ÷ 观察天数。
  - 缺字段降级：没有库存日快照时，不计算缺货天数和缺货率。
  - 注意：如果只有当前库存，只能判断当前是否缺货，不能计算缺货率。

- 平均日销量（items_sold_per_day）[条件]
  - 计算公式：平均日销量 = 售出件数 ÷ 观察天数
  - 缺字段降级：观察天数为 0 或销售数据不完整时，不计算该指标。
  - 注意：用于辅助补货节奏判断，应结合缺货情况一起解释。

## 客户价值指标

- 新客户数（new_customers）[条件]
  - 计算公式：新客户数 = count(distinct 当前周期内首次购买的客户 ID)
  - 缺字段降级：没有客户 ID 或历史订单时，不计算该指标。
  - 注意：判断新客需要知道客户是否在当前周期之前购买过。

- 复购客户数（returning_customers）[条件]
  - 计算公式：复购客户数 = count(distinct 当前周期有购买，且当前周期前已有历史订单的客户 ID)
  - 缺字段降级：没有客户 ID 或历史订单时，不计算该指标。
  - 注意：复购客户数依赖历史数据，不能只看当前周期订单。

- 复购率（repeat_purchase_rate）[条件]
  - 计算公式：复购率 = 当前周期复购客户数 ÷ 当前周期去重购买客户数
  - 缺字段降级：没有客户 ID、历史订单或去重购买客户数为 0 时，不计算该指标。
  - 注意：如果业务需要“严格期间内复购率”，应另行说明口径。

- 人均订单数（orders_per_customer）[条件]
  - 计算公式：人均订单数 = 订单数 ÷ 去重购买客户数
  - 缺字段降级：没有客户 ID 或去重购买客户数为 0 时，不计算该指标。
  - 注意：用于衡量购买频率。

- 客户平均消费额（amount_spent_per_customer）[条件]
  - 计算公式：客户平均消费额 = 销售净额 ÷ 去重购买客户数
  - 缺字段降级：没有客户 ID 或去重购买客户数为 0 时，不计算该指标。
  - 注意：用于判断客户价值质量，应与客单价区分。

- 同期群留存 / 同期群复购（cohort_retention，cohort 指按首购时间等条件划分的一组客户）[条件]
  - 计算公式：第 t 期同期群留存率 = 该同期群中第 t 期有订单的客户数 ÷ 同期群规模。
  - 缺字段降级：没有客户 ID、首购时间或足够历史数据时，不计算该指标。
  - 注意：同期群规模过小时，应标记“样本不足”。

- RFM 客户分层（rfm_segment，RFM 指 Recency / Frequency / Monetary，即最近购买时间、购买频次、消费金额）[条件]
  - 计算公式：分别计算 recency、frequency、monetary 后，按分位数打分并组合成客户分层。
  - 缺字段降级：没有客户 ID 或足够历史订单时，不计算该指标。
  - 注意：RFM 分数是当前业务数据内的相对分层，不是行业绝对标准。

- 客户生命周期价值 / 预测消费额（ltv / predicted_spend，LTV 指 Customer Lifetime Value，客户生命周期价值）[条件]
  - 计算公式：历史客户生命周期价值 = 客户历史累计销售净额；预测消费额 = 历史平均周期消费额 × 预测周期数 × 留存概率。
  - 缺字段降级：没有客户 ID 或历史消费数据时，不计算 LTV；没有预测模型或留存概率时，不计算 predicted_spend。
  - 注意：历史累计消费额不等于预测生命周期价值，必须区分历史口径和预测口径。

## 营销效率指标

- 广告展示量（impressions）[条件]
  - 计算公式：广告展示量 = sum(广告展示量)
  - 缺字段降级：没有广告投放数据时，不计算该指标。
  - 注意：展示量用于衡量覆盖，不代表用户兴趣。

- 广告点击量（clicks）[条件]
  - 计算公式：广告点击量 = sum(广告点击量)
  - 缺字段降级：没有广告点击字段时，不计算该指标。
  - 注意：点击量用于衡量兴趣，但不能单独说明成交质量。

- 点击率（ctr，Click-Through Rate）[条件]
  - 计算公式：点击率 = 广告点击量 ÷ 广告展示量
  - 缺字段降级：没有展示量、点击量或展示量为 0 时，不计算该指标。
  - 注意：点击率高不一定代表转化好，需要结合转化数和 ROAS。

- 广告花费（ad_spend）[条件]
  - 计算公式：广告花费 = sum(广告投放消耗金额)
  - 缺字段降级：没有广告花费字段时，不计算平均点击成本、平均转化成本和广告支出回报率。
  - 注意：广告花费应与广告展示、点击和转化使用同一统计周期。

- 平均点击成本（avg_cpc，CPC 指 Cost Per Click）[条件]
  - 计算公式：平均点击成本 = 广告花费 ÷ 广告点击量
  - 缺字段降级：没有广告花费、点击量或点击量为 0 时，不计算该指标。
  - 注意：用于衡量流量获取成本。

- 广告转化数（conversions）[条件]
  - 计算公式：广告转化数 = sum(广告系统定义下的转化次数)
  - 缺字段降级：没有转化字段时，不计算该指标。
  - 注意：必须说明转化定义，例如下单、支付、注册或加购。

- 平均转化成本（cpa，CPA 可指 Cost Per Action 或 Cost Per Acquisition，即单次转化成本或单次获客成本）[条件]
  - 计算公式：平均转化成本 = 广告花费 ÷ 广告转化数
  - 缺字段降级：没有广告花费、转化数或转化数为 0 时，不计算该指标。
  - 注意：用于衡量成交或获客效率。

- 广告支出回报率（roas，Return on Ad Spend）[条件]
  - 计算公式：广告支出回报率 = 广告带来的转化价值 ÷ 广告花费
  - 缺字段降级：没有广告归因销售额、转化价值或广告花费为 0 时，不计算该指标。
  - 注意：如果归因窗口不明确，应标记口径不清。

## SQL 生成约束

- 只生成只读 SQL。
- 不生成 `INSERT`、`UPDATE`、`DELETE`、`DROP`、`ALTER`、`TRUNCATE` 等写操作。
- 先确认事实表与维表的连接键，再写查询。
- 先确定分析粒度，再聚合；不要聚合后再错误 join。
- 周期比较必须保证口径一致、对象可比。
- 商品分析优先从订单商品明细表出发，不要只使用订单头表。
- 客户分析必须显式处理匿名客户 / 游客客户。
- 退款分析要区分“退款发生时间”和“原订单购买时间”。
- 有多币种时必须先统一汇率口径。
- 有多时区时必须先统一业务时区。
- 任何估算字段都要在 SQL 注释或元数据中标记。
- 所有金额类指标应明确是否含税、是否含运费、是否扣除退款。
- 所有占比类指标必须保留分母。
- 所有转化率指标必须保证分子和分母来自同一时间窗口、同一统计粒度。
- 排名类指标应保留排序依据，例如销售额、销量、毛利或毛利率。
- 对比类指标必须明确当前周期和对比周期。
