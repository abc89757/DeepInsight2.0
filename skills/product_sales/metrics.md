# 商品销售分析指标种类

## 使用原则

- 本文档描述“应该取哪些证据”，不写 SQL 细节
- 每个指标都应标记适用条件：
  - [核心]：商品销售场景高频必看
  - [条件]：只有数据库表结构（schema，指数据库中表、字段、关系等结构信息）支持时才启用
- 如果用户问题不需要，你不必把所有核心指标和条件指标都取齐
- 核心思路：先结果，再结构，再效率，再客户，再库存，再营销

## 全渠道核心经营指标

- 销售总额（gross_sales） [核心]
  - 含义：折扣、退款前的成交商品金额
  - 用途：看规模，辅助识别折扣/退款对净销售的侵蚀
- 销售净额（net_sales） [核心]
  - 含义：扣除折扣、退款、销售折让（allowance，指成交后给予客户的价格减免或补偿）后的销售金额
  - 用途：作为默认“销售结果”主指标
- 订单数（orders） [核心]
  - 含义：完成购买的订单数
  - 用途：判断销售变化来自“单量”还是“单价”
- 售出件数（units_sold） [核心]
  - 含义：商品售出件数
  - 用途：区分“销量变化”和“价格变化”
- 去重购买客户数（unique_customers） [核心]
  - 含义：产生购买的去重客户数
  - 用途：判断增长来自更多客户还是更高频购买
- 毛利润（gross_profit） [条件]
  - 含义：净销售扣除销货成本（COGS，Cost of Goods Sold，指已售商品对应的直接成本）后的利润
  - 用途：判断增长是否有质量
- 毛利率（gross_margin） [条件]
  - 含义：毛利润 / 销售净额
  - 用途：衡量商品或渠道盈利质量
- 贡献利润（contribution_profit） [条件]
  - 含义：毛利润继续扣除履约成本、支付手续费、平台费用、广告费用等可归因成本后的利润
  - 用途：比毛利润更接近真实经营质量
- 净利润（net_profit） [条件]
  - 含义：扣除全部费用、利息、税后的最终利润
  - 用途：只有财务完整成本存在时才使用
  - 注意：如果缺完整成本，不要命名为净利润，而是说清楚这是当前数据下能计算出的净利润

## 价格与客单指标

- 总额口径客单价（aov_gross，Average Order Value，平均订单金额） [核心]
  - 含义：平均每笔订单对应的销售总额
  - 用途：看交易额来自更大的订单还是更多订单

- 净额口径客单价（aov_net，Average Order Value，平均订单金额） [核心]
  - 含义：平均每笔订单对应的销售净额
  - 用途：默认客单价指标

- 净额口径平均售价（asp_net，Average Selling Price，平均销售单价） [核心]
  - 含义：平均每件商品的净售价
  - 用途：区分销售变化来自涨价/降价，还是来自售出件数变化

- 折扣金额（discount_amount） [核心]
  - 含义：订单或商品行中的折扣总金额
  - 用途：衡量让利规模

- 折扣率（discount_rate） [核心]
  - 含义：折扣金额占销售总额的比例
  - 用途：判断销售增长是否依赖降价或促销让利

- 退款金额（refund_amount） [核心]
  - 含义：已退还给客户的金额
  - 用途：判断成交质量与售后风险

- 金额口径退款率（refund_rate_amt） [核心]
  - 含义：退款金额 / 销售总额
  - 用途：衡量退款对销售结果的侵蚀

- 件数口径退货率（return_rate_units） [条件]
  - 含义：退货件数 / 售出件数
  - 用途：定位商品质量、商品描述不准确或用户预期不符等问题

## 商品结构指标

- 单品销售贡献占比（sku_sales_share，SKU 指 Stock Keeping Unit，即库存量单位/单品编码）[核心]
  - 含义：某个单品销售额占整体销售额的比例。
  - 用途：寻找销售拉动项和拖累项。

- 类目销售贡献占比（category_sales_share）[核心]
  - 含义：某个商品类目的销售额占整体销售额的比例。
  - 用途：观察品类结构变化。

- 品牌销售贡献占比（brand_sales_share）[条件]
  - 含义：某个品牌的销售额占整体销售额的比例。
  - 用途：观察品牌结构迁移。

- 单品毛利排序（sku_margin_rank，SKU 指库存量单位/单品编码）[条件]
  - 含义：按单品毛利润或毛利率进行排序。
  - 用途：避免只看销量冠军，忽略商品盈利能力。

- 商品结构变化（product_mix_change）[核心]
  - 含义：商品销售结构是否偏向高价/低价、高毛利/低毛利商品。
  - 用途：解释客单价、平均售价和毛利率变化。

## 渠道 / 门店 / 地区指标

- 渠道销售额 / 门店销售额 / 地区销售额（channel_sales / store_sales / region_sales）[核心]
  - 含义：不同渠道、门店、地区对应的销售结果。
  - 用途：寻找增长来源与问题集中区域。

- 渠道毛利 / 门店毛利（channel_margin / store_margin）[条件]
  - 含义：不同渠道、门店对应的毛利水平或毛利率。
  - 用途：比较“销售规模”和“盈利质量”是否一致。

- 渠道订单占比（channel_order_share）[核心]
  - 含义：不同渠道订单数占总订单数的比例。
  - 用途：解释销售变化是来自订单量变化，还是来自客单价变化。

- 可比门店销售增长率（same_store_sales_growth，也常称同店销售增长率）[条件]
  - 含义：在可比门店范围内，当前周期相对对比周期的销售增长。
  - 用途：实体零售高价值指标；需要明确定义哪些门店属于可比门店。

## 线上漏斗指标

- 商品浏览量（items_viewed）[条件]
  - 含义：商品详情页被浏览的次数或商品被浏览的件数。
  - 用途：衡量商品被用户看到的机会。

- 加购商品数（items_added_to_cart）[条件]
  - 含义：被加入购物车的商品数量。
  - 用途：衡量用户购买意向。

- 进入结账商品数（items_checked_out）[条件]
  - 含义：进入结账流程的商品数量。
  - 用途：定位支付前流失问题。

- 购买次数（purchases）[条件]
  - 含义：完成购买的次数。
  - 用途：衡量最终成交转化。

- 浏览到加购转化率（view_to_cart_rate）[条件]
  - 含义：加购商品数 / 商品浏览量。
  - 用途：判断商品详情页、价格、商品吸引力是否存在问题。

- 加购到结账转化率（cart_to_checkout_rate）[条件]
  - 含义：进入结账商品数 / 加购商品数。
  - 用途：判断购物车到下单阶段的损耗。

- 结账到购买转化率（checkout_to_purchase_rate）[条件]
  - 含义：购买商品数 / 进入结账商品数。
  - 用途：判断支付流程、运费、库存可用性、履约承诺等是否造成流失。

- 浏览到购买转化率（view_to_purchase_rate）[条件]
  - 含义：购买商品数 / 商品浏览量。
  - 用途：衡量商品级综合转化效率。

## 促销指标

- 促销曝光量 / 促销点击量（promo_views / promo_clicks）[条件]
  - 含义：促销位被展示的次数与被点击的次数。
  - 用途：判断促销位是否被看到，以及是否足够吸引用户。

- 促销订单数 / 促销销售额（promo_orders / promo_sales）[条件]
  - 含义：与促销活动相关的订单数和销售额。
  - 用途：判断促销活动的覆盖范围与销售产出。

- 促销销售占比（promo_sales_share）[条件]
  - 含义：促销相关销售额占整体销售额的比例。
  - 用途：判断整体销售结果对促销活动的依赖程度。

- 促销毛利影响（promo_margin_impact）[条件]
  - 含义：促销活动对毛利润或毛利率的影响。
  - 用途：判断是否存在“有销量、无利润”的风险。

## 库存与供应链指标

- 期末库存件数 / 期末库存成本（closing_inventory_units / closing_inventory_cost）[条件]
  - 含义：期末仍在库的商品件数和对应成本。
  - 用途：判断压货、断货和现金占用风险。

- 库存周转次数（inventory_turnover）[条件]
  - 含义：一定周期内库存被销售并替换的次数。
  - 用途：判断库存使用效率。

- 售罄率 / 动销消化率（sell_through_rate）[条件]
  - 含义：售出数量 / 收货数量。
  - 用途：判断商品卖得快不快。

- 库存可支撑天数（days_on_hand，DOH，Days on Hand）[条件]
  - 含义：按照当前销售速度，现有库存还能支撑的天数。
  - 用途：判断压货与缺货风险。

- 缺货天数 / 缺货率（stockout_days / stockout_rate）[条件]
  - 含义：商品处于缺货状态的天数，或缺货天数占观察周期的比例。
  - 用途：判断销售差是因为“卖不动”，还是因为“没货卖”。

- 平均日销量（items_sold_per_day）[条件]
  - 含义：观察周期内平均每天售出的商品数量。
  - 用途：辅助判断补货节奏。

## 客户价值指标

- 新客户数（new_customers）[条件]
  - 含义：首次购买的客户数量。
  - 用途：观察拉新效果。

- 复购客户数（returning_customers）[条件]
  - 含义：已有历史购买记录，并在当前周期再次购买的客户数量。
  - 用途：观察留存质量。

- 复购率（repeat_purchase_rate）[条件]
  - 含义：复购客户数 / 去重购买客户数。
  - 用途：衡量客户复购能力。

- 人均订单数（orders_per_customer）[条件]
  - 含义：平均每位购买客户产生的订单数量。
  - 用途：衡量购买频率。

- 客户平均消费额（amount_spent_per_customer）[条件]
  - 含义：平均每位购买客户贡献的销售净额。
  - 用途：判断客户价值质量。

- 同期群留存 / 同期群复购（cohort_retention，cohort 指按首购时间等条件划分的一组客户）[条件]
  - 含义：按首购时间分组后，不同客户批次在后续周期的留存或复购情况。
  - 用途：判断不同批次客户的后续表现。

- RFM 客户分层（rfm_segment，RFM 指 Recency / Frequency / Monetary，即最近购买时间、购买频次、消费金额）[条件]
  - 含义：按最近购买时间、购买频次和消费金额对客户分层。
  - 用途：识别高价值客户、沉睡客户和流失风险客户。

- 客户生命周期价值 / 预测消费额（ltv / predicted_spend，LTV 指 Customer Lifetime Value，客户生命周期价值）[条件]
  - 含义：客户在整个生命周期内可能贡献的价值，或模型预测的未来消费水平。
  - 用途：更偏中长期客户价值判断。

## 营销效率指标

- 广告展示量（impressions）[条件]
  - 含义：广告被展示的次数。
  - 用途：观察广告覆盖范围。

- 广告点击量（clicks）[条件]
  - 含义：广告被点击的次数。
  - 用途：观察用户兴趣。

- 点击率（ctr，Click-Through Rate）[条件]
  - 含义：广告点击量 / 广告展示量。
  - 用途：衡量广告、素材或关键词与用户需求的相关性。

- 广告花费（ad_spend）[条件]
  - 含义：广告投放消耗的费用。
  - 用途：衡量获客投入。

- 平均点击成本（avg_cpc，CPC 指 Cost Per Click）[条件]
  - 含义：广告花费 / 广告点击量。
  - 用途：衡量流量获取成本。

- 广告转化数（conversions）[条件]
  - 含义：广告系统定义下的转化次数。
  - 用途：衡量投放产出。

- 平均转化成本（cpa，CPA 可指 Cost Per Action 或 Cost Per Acquisition，即单次转化成本或单次获客成本）[条件]
  - 含义：广告花费 / 转化数。
  - 用途：衡量成交或获客效率。

- 广告支出回报率（roas，Return on Ad Spend）[条件]
  - 含义：广告带来的转化价值 / 广告花费。
  - 用途：衡量广告投放是否值得。

## 取证优先级建议

默认优先级：

1. 销售净额（net_sales）/ 订单数（orders）/ 售出件数（units_sold）/ 净额口径客单价（aov_net）
2. 折扣、退款、毛利润（gross_profit）/ 毛利率（gross_margin）
3. 商品、渠道、门店、地区结构
4. 线上漏斗 或 库存效率，根据场景二选一，也可以同时启用
5. 客户复购 / 同期群分析（cohort analysis）/ RFM 客户分层
6. 广告效率

## 缺字段时的降级策略

- 没有成本字段：不输出毛利润（gross_profit）、毛利率（gross_margin）、贡献利润（contribution_profit）、净利润（net_profit）。
- 没有事件埋点数据：不输出浏览到加购转化率（view_to_cart_rate）、结账到购买转化率（checkout_to_purchase_rate）等漏斗指标。
- 没有库存快照（inventory snapshots，指某一时间点的库存状态记录）：不输出库存周转次数（inventory_turnover）和库存可支撑天数（DOH），只保留库存余额类指标。
- 没有客户 ID（customer_id）：不输出复购、同期群分析（cohort analysis）和 RFM 客户分层。
- 没有广告花费（ad_spend）：不输出点击率（CTR）、平均点击成本（CPC）、平均转化成本（CPA）和广告支出回报率（ROAS）。
