# 分组内Top N分析场景总说明

本 skill 适用于需要在某个业务分组维度内，识别组内表现最优的个体或子项的数据分析场景。

## 场景适用范围

当用户的分析目标明确是“分组内”查找时，应优先考虑本场景。其核心特征是分析目标并非全局排名，而是**分组内排名**。

适用条件包括：
- 用户问题涉及“各XX下最好的YY”、“每个XX中排名前N的YY”等表述。
- 分析目标是识别在某个父维度（如品牌、品类、渠道、地区）下的子维度（如具体商品、门店、用户群）在关键结果指标（如销量、销售额、利润）上的最佳表现。
- 典型业务场景包括：电商平台各品类下销量最高的商品、各门店销售额最高的产品、各年龄段用户消费额最高的渠道、各地区表现最好的销售员等。

## 这个场景要回答什么

分组内Top N分析不只回答“谁是第一”，而是要回答：

- 核心排名结果：在给定分组维度下，哪个子项在核心衡量指标上表现最好？
- 结果的稳健性：这种“最好”的表现是绝对领先还是微弱优势？是否稳定？
- 指标的全面性：“最好”的定义是否准确？单一指标（如销量）下的第一，是否在利润率、库存周转等其他维度上也表现优秀？是否存在“以量取胜”但质效不高的情况？
- 分组的可比性：各分组的规模是否具有可比性？一个很小分组内的“第一”是否具有与主流分组同等重要的业务意义？

## 默认分析框架

执行此类分析，应遵循以下可复用路径：

1.  **定义分析范围与口径**
    - 明确时间范围。
    - 明确核心衡量指标（如成交量、销售额）及其计算口径（如是否含退货、是否含折扣）。
    - 如有需要，限定分析对象状态（如已完成交易、在售状态等）。

2.  **确认分组维度与子项维度**
    - 清晰定义用于分组的“父维度”（如品牌、品类）。
    - 清晰定义用于排名的“子维度”（如具体车型、商品SKU）。
    - 确保数据同时包含这两个维度的标识字段和用于排序的度量值字段。

3.  **获取分组总体概况**
    - 了解父维度的分布情况，例如总共有多少个分组，各分组的总体规模（如总销量、总销售额）如何。
    - 此步骤有助于判断后续分析的侧重点，避免在业务上不具代表性的过小分组上投入过多精力。

4.  **执行分组内Top排序**
    - 在每个“父维度”组内，按核心指标（如销量）降序排列“子维度”。
    - 根据分析目标，找出排名第一或前N名（如Top 3, Top 5）的子项。
    - 建议在得出核心排名后，检查排名靠前的子项在辅助指标（如销售额、利润）上的表现，以形成更立体的判断。

## 触发继续追证的条件

满足以下任一条件时，应优先继续追证或细化分析，而非直接输出Top 1结论：

- **分组规模差异显著**：当大多数分组规模很小，少数分组规模巨大时，需判断结论是服务于全局理解还是特定分组洞察，必要时应分别呈现。
- **子项分布高度集中**：如果某个子项在组内占比（如销量占比）绝对领先（如超过50%），应提示此“一超”格局；如果多个子项指标值非常接近，应提示“并列”或“胶着”状态。
- **核心指标定义存疑**：用户对“卖得最好”的定义模糊，或业务上存在多个关键指标（如销量、销售额、毛利）且排名结果不一致时，需明确主次或提供多维排名。
- **数据存在明显异常**：排名结果出现单价异常低、销量与历史趋势严重背离等情况，需核实数据或提示异常。
- **需要探究背后原因**：当用户不仅想知道“是什么”，还想知道“为什么”该子项能成为Top时，需要引入更多维度（如价格、促销、评价、上架时间）进行归因分析。

## 必须避免的行为

- **避免混淆分组维度与全局排名**：用户的分析目标是“各品牌内”最好的车型，而非“所有品牌中”最好的车型。必须严格遵循“分组内排序”这个前提，不得将分组内排名误用为全局排名。
- **避免用单一指标武断定义“最好”**：即使用户只提及“销量”，也应考虑业务实质，提示撰写人是否需要结合利润、库存等指标综合判断“好”的标准。除非数据或业务明确只关注单一指标。
- **避免忽略分组规模的业务重要性**：在呈现结论时，应避免将销量极少的分组下的“第一名”与主流大分组的“第一名”并列讨论，除非另有说明。需要明确分组的代表性。
- **避免使用具体字段名作为硬性要求**：数据要求应抽象为“需要包含分组维度标识、子项维度标识和核心度量值”，字段名（如`brand`、`model`、`sales_volume`）仅为常见示例。
- **避免直接套用本次任务的具体结论**：如“某品牌的某车型销量第一”是具体结论，而非可复用的分析规则。

## 输出目标

你的输出应至少包含：
- 本轮分析目标（明确为分组内Top N查找）。
- 明确的分组维度与子项维度。
- 需要的关键数据字段（分组标识、子项标识、核心度量值）。
- 是否需要考虑辅助排名指标。
- 对分组规模和排名结果分布的初步判断。
- 是否触发继续追证条件，还是已可基于现有证据给出排名结论。
